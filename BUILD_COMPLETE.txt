================================================================================
                           QUICK START (5 MIN)
================================================================================

Step 1: Install
  $ pnpm install

Step 2: Create .env.local with Firebase credentials
  NEXT_PUBLIC_FIREBASE_API_KEY=...
  NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=...
  NEXT_PUBLIC_FIREBASE_PROJECT_ID=...
  NEXT_PUBLIC_FIREBASE_DATABASE_URL=...
  (See QUICK_START.md for all env vars)

Step 3: Add sample data to Firebase
  - Go to Firebase Console
  - Create Realtime Database
  - Import sample products JSON
  (See FIREBASE_SETUP.md for details)

Step 4: Run dev server
  $ pnpm dev

Step 5: Visit http://localhost:3000/shop
  - See products!
  - Add to cart!
  - Experience real-time magic!
